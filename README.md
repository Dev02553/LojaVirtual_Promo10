projeto final de Semestre, utilizando 4 desing patterns

1-Factory Method
2-Observer Pattern
3-Strategy Pattern

a adição dos itens, precificação e tipificação, além de adicionar ou deletar itens do carrinho, alpem de deletar um ou limpar o
carrinho
